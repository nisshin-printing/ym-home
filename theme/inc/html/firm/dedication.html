事務所理念

  • 法律のプロ集団として，質の高いサービスを心がけます
  • 所員全員が，教養と専門的知識を深めるため，日々研鑽に努めます
  • 公益活動を通じて，社会貢献に努めます

ロゴマークに込めた想い

シンボルマーク　コンセプト

「笑顔・闘志・信頼」を掲げて

ロゴマークは、山下江法律事務所の「山」の字をモチーフに「笑顔」を表しています。あなたの人生を応援する法律事務所でありたい。依頼者に「笑顔」を届けるために、全力を。そして、人と人が繋がっているようにも見える図柄で中四国最大級である法律事務所としての「信頼」を表しています。

メインカラー　コンセプト

山下江法律事務所のカラーは、「闘志」の色。

弁護士という仕事には「闘志」がなければならない。依頼者が命がけで取り組んでいる以上、こちらも命をかけて取り組む。「闘志」をもってできる限りのあらゆることを全力でやる。もし負けた場合でも、この人に頼んでよかったと思ってもらえるように。

所長　山下江

事務所理念に基づき所員が積極的に実践すべきこと

事務所理念に基づき、それの具現化に向け、全所員一丸となり邁進してまいります。

依頼者のみなさまへのお約束

「親切な相談」・「適切な解決」をモットーに、「全力投球」で取り組みます

  1. 分かりやすい説明を心がけます
  2. 解決のための具体的選択肢をなるべく多く提示し、依頼者様の気持ちを汲んだ解決方法を目指します
  3. 受任時には、明確な費用説明を行い、可能な限りの見通しを示します
  4. すみやかな対応を心がけ、事案の進展状況を適宜報告します

地域のみなさまへのお約束

地域社会の健全な発展に貢献し、その活性化に積極的に関わります

  1. 反社会的勢力からの依頼を受任せず、毅然とした態度で対決いたします
  2. 業務を通じて、不正・差別のない社会の実現に寄与します
  3. 企業のコンプライアンス（法令遵守）を支援し、公正な経済活動の行われる社会の実現に寄与します
  4. ＣＳＲ（企業の社会的責任）を重視し、自らも地域活動に積極的に参加します

事務所員への約束

所員一人一人の個性を尊重し、所員が自己実現できる職場環境と教育制度を整えます

  1. 所員全員が安心して楽しく働ける職場を作ります
  2. 労働法規を遵守し、育児・介護など各所員が抱える個別の状況に向き合います。
  3. 個々の潜在能力を最大限に伸ばす教育をし、所内でキャリアアップする体制を整えます
  4. 仕事と生活の調和（ワークライフバランス）の実現を目指します

自分への約束

日々の研鑽により、自己の成長を実現します

  1. 法律以外の分野の知識習得にも積極的にチャレンジします
  2. 何事にも疑問をもち、自分の頭で考えます
  3. プラス思考を心がけます
  4. いつも感謝の念を忘れません
