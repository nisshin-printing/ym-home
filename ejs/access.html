広島駅からお越しの方

広島駅からバスでお越しの方

広島駅南口からバスで当事務所まで①

  • 所要時間 : 約20分
  • 料金 : 160円

印刷用PDF

広島交通高陽B団地線（30号線・7番乗り場）に乗車してください。「女学院前駅」で下車し、広島電鉄（市電）白島線沿いに白島方面（北）へ直進すると「縮景園前駅」があるので、そこから西へ約100m進み、最初の信号（検察庁東南）を左折。当事務所のビル（上八丁堀ビル）は左側にございます。
※縮景園前駅から徒歩約1分（距離： 約102m）です。

広島駅南口からバスで当事務所まで②

  • 所要時間 : 約15分
  • 料金 : 160円

印刷用PDF

乗り場は広島駅南口から広島東郵便局方面へ向かい、居酒屋がたくさん入ったビルのある方へ横断歩道を渡り、1階に「すしざむらい」という居酒屋が入ったビルの目の前です。広島バス旭町線（26号線・21番乗り場）に乗車し、「女学院前駅」で下車してください。広島電鉄（市電）白島線沿いに白島方面（北）へ直進すると「縮景園前駅」があるので、そこから西へ約100m進み、最初の信号（検察庁東南）を左折。当事務所のビル（上八丁堀ビル）は左側にございます。
※縮景園前駅から徒歩約1分（距離： 約102m）です。

広島駅から市内電車でお越しの方

広島駅南口から広島電鉄（市電）で当事務所まで

  • 所要時間 : 約30分
  • 料金 : 160円（乗り換え有り）

印刷用PDF

市電5番以外に乗車して、「八丁堀駅」で下車してください。その後白島線へ乗り換えをし、「縮景園前駅」で下車してください。このとき、乗り換えである旨を車掌へお伝えください。縮景園前駅からは、西へ約100m進み、最初の信号（検察庁東南）を左折。当事務所のビル（上八丁堀ビル）は左側にございます。
※縮景園前駅から徒歩約1分（距離： 約102m）です。

広島バスセンターからお越しの方

広島バスセンターからバスでお越しの方

  • 所要時間 : 約10分
  • 料金 : 160円

印刷用PDF

「広島駅行き」に乗車（ただし、「合同庁舎前駅」経由のもの）し、「合同庁舎前駅」で下車。そのまま東（バスが進んでいた方向）へ直進し、下車後2つめの角（服飾学院が左手にある）を左に曲がる。そのまま直進し、左手側にセブン-イレブンを過ぎて、「うどんのちから」まで進む。「うどんのちから」の前の横断歩道を進行方向右手に曲がる。「花かぶき」という雑貨屋が入っているグレーのビルが当事務所です。

バスセンターから徒歩で当事務所まで

  • 所要時間 ： 約20分

バスでお越しの方のための地図に記載されている目印を参考にしてください。

  1. 県庁、市民病院側（東）へ横断歩道を渡り、北へ直進
  2. 県庁、市民病院を右手側に見て直進し、突き当りの大きい道路（城南通り）を、右折して進む
  3. ガソリンスタンドを過ぎ、右手に「法テラス」が見えたらそこの横断歩道を進行方向に向かって左（服飾学院の方）に渡る
  4. そのまま直進し，進行方向左手側のセブンイレブンを過ぎて，「うどんのちから」まで進む
  5. うどんのちからの前の横断歩道を進行方向右手に曲がる
  6. 「花かぶき」という雑貨屋が入っているグレーのビルが当事務所（向かって右手に自動ドアの入口）

自転車・バイクでお越しの方

当事務所のビルには駐輪場がありません。近くに市営の基町駐輪場（有料）・民営の路上駐輪場（八丁堀・鉄砲町）がございますので、そちらをご利用ください。徒歩約７分から９分（５００ｍから７００ｍ）の距離です。
